# nvPress概述

nvPress是一个基于nodejs实现的内容管理系统。

功能包括：用户管理、媒体管理、文章管理、主题、插件等。

用于：个人博客、作品集、网站内容展示。

可通过主题、插件来实现功能的二次扩展。
